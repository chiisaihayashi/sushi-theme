====

## Install

After installing [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh),  install this by:

```bash
$ curl -fsSL https://github.com/chiisaihayashi/sushi-theme/blob/master/sushi.zsh-theme >> ~/.oh-my-zsh/themes/sushi.zsh-theme
```

And configure `.zshrc`:

```bash
ZSH_THEME="sushi"
```
